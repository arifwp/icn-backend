## Installation

Install my-project with npm

```bash
# clone repository
$ git clone https://github.com/arifwp/icn-backend

# move to project folder
$ cd icn-backend

# configure env
$ rename .env.example to .env

$ npm Install

# watch mode
$ npm run start:dev
```
